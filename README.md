## CHaser_C#
CHaserクライアントのC#テンプレートです。

広く使われているらしい C# のライブラリがどこにも見当たらない`>_<`ので、

広く使われている [Python のライブラリ](https://github.com/tomio2480/chaser-python)に合わせて作りました。
